# 任务列表测试

## 基础功能

- [ ] 未完成的任务1
- [x] 已完成的任务1
- [ ] 未完成的任务2
- [X] 已完成的任务2（大写X）

## 嵌套任务列表

- [ ] 父任务1
  - [ ] 子任务1.1
  - [x] 子任务1.2（已完成）
  - [ ] 子任务1.3
- [x] 父任务2（已完成）
  - [x] 子任务2.1
  - [x] 子任务2.2

## 混合列表

- 普通列表项1
- [ ] 任务列表项1
- 普通列表项2
- [x] 任务列表项2（已完成）
- 普通列表项3

- [ ]
- [ ]
- [ ] 有内容的任务
