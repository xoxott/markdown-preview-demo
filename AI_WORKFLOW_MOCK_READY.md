## 🚀 快速开始

### 1. 启动项目
```bash
npm run dev
```

### 2. 访问工作流页面
```
http://localhost:9527/ai-workflow
```

### 3. 体验功能
1. **查看列表**：3个示例工作流
2. **编辑工作流**：点击"编辑"进入可视化编辑器
3. **创建新工作流**：从节点库拖拽节点到画布
4. **执行工作流**：点击"执行"查看运行结果
5. **查看历史**：查看执行日志和版本历史

## 🎮 交互操作

### 画布操作
- **缩放**：鼠标滚轮
- **平移**：右键拖拽
- **选择节点**：左键点击
- **多选**：Ctrl/Cmd + 点击
- **删除**：Delete 或 Backspace
- **全选**：Ctrl/Cmd + A

### 节点操作
1. 从左侧节点库拖拽节点到画布
2. 拖拽节点的输出端口到另一个节点的输入端口
3. 点击节点查看右侧配置面板
4. 修改配置后点击"保存配置"
5. 点击工具栏"保存"按钮保存工作流
