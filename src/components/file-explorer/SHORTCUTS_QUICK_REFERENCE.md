# ⌨️ 快捷键快速参考

## 文件操作

```
Ctrl+A          全选文件
Ctrl+C          复制
Ctrl+X          剪切
Ctrl+V          粘贴
Delete          删除
F2              重命名
Enter           打开
Alt+Enter       属性
```

## 视图切换

```
Ctrl+1          网格视图
Ctrl+2          列表视图
Ctrl+3          平铺视图
Ctrl+4          详细视图
Ctrl+5          内容视图
```

## 其他

```
F5              刷新
Ctrl+Shift+N    新建文件夹
Escape          取消选择
```

---

💡 **提示**: 点击文件管理器区域使其获得焦点后，快捷键即可生效。
